Extracted tile graphics from the ROM data section

These graphics assets were extracted from the ROM's data section using custom decompression tools that implement the game's Huffman+LZ77 pipeline (matching the GBA BIOS HuffUnComp and LZ77UnCompWram syscalls). Assets are rendered as 4bpp tile sheets colorized with their closest matching ROM palette. Actual in-game colors may vary as palettes are selected dynamically at runtime.

Compression Format

The game uses a two-stage compression pipeline implemented in FUN_08043af4 (DecompressData):

  • Header word (4 bytes): Bit 31 = Huffman flag, bits 0-30 = decompressed size
  • If bit 31 set: 4-bit Huffman (HuffUnComp / SWI 0x13) followed by LZ77 (LZ77UnCompWram / SWI 0x11)
  • If bit 31 clear: LZ77 only
