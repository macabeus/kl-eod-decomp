Sound Engine Documentation

Nintendo MusicPlayer2000 ("Sappy" / "m4a")

Klonoa: Empire of Dreams uses the standard GBA sound engine known as Nintendo MusicPlayer2000, commonly referred to as "Sappy" or "m4a" in the decomp community. This is the same engine found in Pokemon, Kirby, Fire Emblem, and most first-party and many third-party GBA titles.

ROM Layout

Region Address Range Size Content
Sound tables 0x08117xxx ~3 KB Pitch, frequency, command dispatch LUTs
Voice tables 0x08117CBC – 0x081189xx ~3 KB 3 instrument/voice group tables
PCM samples 0x0811BB00 – 0x08163044 ~300 KB Signed 8-bit PCM waveform data
Song/track data 0x0816CB04 – 0x08184604 ~100 KB Song headers + bytecode track data
Song pointer table 0x08118AE4 ~1.3 KB 165 entries (39 songs + 126 SFX)

Structure

Song Table

Located at ROM_MUSIC_META_TABLE = 0x08118AE4. Contains 165 entries of the form: 

struct SongTableEntry {
    u32 songHeaderPtr;
    u32 flags;  // songs: 0, SFX: encodes priority
};

Song Header

struct SongHeader {
    u8  numTracks;
    u8  blockCount;
    u16 priority;
    u32 voiceGroupPtr;
    u32 trackPtrs[numTracks];
};

Track Data

Proprietary MIDI-like bytecode. Encodes notes (0x81–0xB0), waits (0x00–0x7F), and control commands (tempo, instrument select, volume, pan, pitch bend, loops, subroutines).

Voice / Instrument Entry (12 bytes)

struct VoiceEntry {
    u8  type;       // see Voice/Instrument Types table
    u8  baseKey;
    u16 flags;
    u32 samplePtr;
    u32 adsr;
};

PCM Sample Header (16 bytes)

struct SampleHeader {
    u32 flags;
    u32 pitch;
    u32 loopOffset;
    u32 length;
    // followed by signed 8-bit PCM data
};

m4a Bytecode Reference

Byte Range Meaning
0x00 – 0x7FWait / duration values
0x80Rest
0x81 – 0xB0Note on (MIDI note number)
0xB1End of track (FINE)

0xBFSet panning

Tools

The m4a format is compatible with standard GBA music tools:

  • sappy — GUI song player/editor
  • mid2agb — MIDI to m4a converter (part of the official Nintendo SDK)
  • agb2mid — m4a to MIDI converter
  • pokemid — MIDI conversion tool for Pokemon games (works with any m4a game)
